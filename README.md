# ⏰ Clock Suite & AIML Assistant

A modern desktop productivity application built with Python Tkinter that combines a complete clock utility suite with a custom AIML-powered conversational assistant.

The application provides alarm management, world clocks, stopwatch tracking, countdown timers, and chatbot-based voice-style command execution through a custom AIML engine.

---

## Features

### Alarm Management
- Create, edit, enable, and disable alarms
- Repeat scheduling by day
- Snooze configuration
- Sound and vibration preferences
- Alarm sorting and management

### World Clock
- Track multiple cities across different time zones
- Real-time clock synchronization
- Custom city offset support
- Detailed city time information

### Stopwatch
- High-precision stopwatch
- Lap tracking
- Session history storage
- Performance timing analysis

### Countdown Timer
- Preset timers
- Custom countdown creation
- Circular progress visualization
- Pause, resume, and cancel functionality

### AIML Chat Assistant
- Custom AIML pattern matching engine
- Natural language command processing
- Alarm creation through chat commands
- World clock management via chatbot
- Action-driven conversational interface

### Responsive Interface
- Mobile-style phone layout
- Desktop dashboard layout
- Custom UI components
- Smooth animations and modern design

### Persistent Storage
- JSON-based local data storage
- Automatic state saving
- User preferences retention
